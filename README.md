## Default Variables

### sanoid_datasets

List of datasets

#### Default value

```YAML
sanoid_datasets: []
```

#### Example usage

```YAML
sanoid_datasets:
  - name: rpool/USERDATA
    use_template: general
    recursive: True
    process_children_only: True
```

### sanoid_templates

List of templates

#### Default value

```YAML
sanoid_templates:
  - name: general
    frequently: 0
    hourly: 24
    daily: 7
    monthly: 12
    yearly: 0
    autosnap: true
    autoprune: true
```
